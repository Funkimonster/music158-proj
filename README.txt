Open Final.maxpat. 
Start the window (at the bottom) and adjust the two channel
volume sliders.
Adjust the dropdown melody blocks to your liking, 
then press the Blue button then the Red button.

The Blue button loads up the new melody and readies it for playback.
The Red button will interrupt the current playback and 
restart the melody with any changes you've made.
